# plugin-number-line-task

## Overview

This will be a plugin to create the Cohen-Ray Number Line.

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/plugin-number-line-task">
```

### Via NPM

```
npm install @jspsych-contrib/plugin-number-line-task
```

```js
import jsPsychPluginNumberLineTask from "plugin-number-line-task";
```

## Compatibility

`@jspsych-contrib/plugin-number-line-task` requires jsPsych v8.0.0 or later.
